1) Меняешь файл/файлы
2) переходишь через cmd в директорию с проектом
   cd /d  D:/git/pyython
3) git status - проверить состояния изменний
4) git add test.txt - добавить только файл test.txt
   git add . - добавить все измененные файлы
5) git commit -m "любой текст с описание имсправления"
6) git push - перенести все локальные изменения на сервер


git pull - обновить все из удаленного репозитория.
